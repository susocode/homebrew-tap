# susocode/homebrew-tap

Homebrew tap for [susocode](https://github.com/susocode) tools.

## Install

```bash
brew tap susocode/tap
brew install ai-frames
```

## Tools

| Formula | Description |
|---|---|
| `ai-frames` | Centralized AI context manager for multi-repo workspaces |

## Usage

```bash
# Launch the UI
ai-frames

# Sync context to workspace (no UI)
ai-frames install
```

## Update

```bash
brew upgrade ai-frames
```

## Source

- CLI repo: [susocode/ai-frames-cli](https://github.com/susocode/ai-frames-cli)
- Website: [ai-frames.org](https://ai-frames.org)
